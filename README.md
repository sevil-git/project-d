## DASHBOARD IMAGE PLAN

Use one very small visual per card or section. Keep them quiet, low-saturation, and slightly hidden in a corner so they support the UI without becoming the focus.

### Minimal style rules

- Use thin line art, soft gradients, or almost-flat monochrome shapes.
- Keep opacity low, around 8% to 18% if used as background art.
- Avoid glossy 3D, bright glows, or busy textures.
- Place the artwork in a corner, edge band, or header side area.
- Prefer transparent backgrounds where possible.

## GENERATION RECOMMENDATIONS

### Tools & Platforms
1. **Midjourney** - Best for high-quality, cohesive fintech aesthetics
2. **DALL-E 3** - Good for specific icons and custom graphics
3. **Stable Diffusion** - Best for batch generation and fine-tuning
4. **Adobe Firefly** - Great for integration with design workflows

### Tips for Best Results
1. Use consistent seed values for visual cohesion
2. Specify "fintech aesthetic," "luxury," "modern" in prompts
3. Always mention target color scheme in prompts
4. Request "professional," "enterprise-grade," or "premium" for trust
5. Include negative prompts: "no pixelation, no blurry elements, no cartoon"
6. Test aspect ratios and request upscaling for crisp details
7. Generate 3-5 variations per concept, then refine

### Estimated Timeline
- Academy section: 2-3 days (9 track icons + tier visualization)
- Unilevel/Plan section: 2 days (tree diagram + examples)
- KYC/Passport: 1-2 days (card design + flow)
- Timeline/Roadmap: 1-2 days
- Dashboard graphics: 3-4 days (icons + visualizations)
- **Total: ~2-3 weeks** for complete visual overhaul
